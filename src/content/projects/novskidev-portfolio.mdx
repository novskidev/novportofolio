Novskidev Portfolio adalah website personal untuk mengumpulkan profil, project notes, blog, dan kontak Novian Nadari dalam satu tempat.

## Tujuan

Project ini dibuat agar identitas digital saya lebih rapi dan mudah dikembangkan. Setiap bagian website dirancang untuk bisa tumbuh perlahan: halaman proyek, tulisan teknis, catatan singkat, dan halaman about.

## Stack

- Astro untuk static site generation.
- MDX untuk konten blog dan project notes.
- Pagefind untuk pencarian statis.
- Bun untuk workflow development dan build.
